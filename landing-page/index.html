WorldSurf App Testing

Project Description

This page is designed to act as a landing page for all test builds of the WorldSurf application for the Web Interface and Usability class at Full Sail University. During the course of the month, multiple builds will be added to the site via the links below. The tests will be created with the assistance of Navflow and the images through the use of Adobe Illustrator. The images created through Illustrator will simulate a high-fidelity wireframe for the WorldSurf application.
